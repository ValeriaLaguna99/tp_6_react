# TP6 · React.js

Aplicación React con componentes, estado y listas.

## Tecnologías
- React
- Vite
- JSX
- CSS

## Instrucciones
npm install
npm run dev

## Componentes
- Header y Footer
- Card reutilizable con props

- Formulario controlado con preview en vivo
- Lista de productos con filtrado
- To-Do App con CRUD completo
